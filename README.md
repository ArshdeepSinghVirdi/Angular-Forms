## (Local Setup)

### Prerequisites
- **Node.js** (v18 or higher)
- **npm** or **yarn**

### Installation Steps

```bash
# 1. Navigate to the frontend folder
cd frontend

# 2. Install dependencies
yarn install
# OR
npm install

# 3. Start the development server
npm run start

# 4. Open your browser
The app will automatically open at http://localhost:3000
```
